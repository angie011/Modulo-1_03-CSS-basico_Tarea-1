Pseudoclases

Son estilos especiales que se utilizan para dar estilos a elementos respecto al
comportamiento que experimentan en determinado momento.

Ejemplo

.defClases p:hover{
background-color: lawngreen; text-align: justify;
}

Pseudoelementos

Son estilos que permiten hacer referencia a comportamientos virtuales no tangibles, es decir, a elementos que no existen realmente en el HTML y que se pueden generar desde CSS.

Ejemplo

.defElementos p:hover{
background-color: yellow; text-align: justify;
}
